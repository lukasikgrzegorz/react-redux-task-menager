#  Task Menager

Simple task menager for redux practice.

https://lukasikgrzegorz.github.io/react-redux-task-menager

## 🚀 Run Locally

Clone the project

```bash
  git clone https://github.com/lukasikgrzegorz/react-redux-task-menager
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## 📝 Tech Stack

React, Redux
